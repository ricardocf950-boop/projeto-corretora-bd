# 📈 Sistema de Banco de Dados - Corretora de Investimentos

Este repositório contém o projeto de desenvolvimento do banco de dados relacional para uma corretora de ações e investimentos, desenvolvido para a matéria de Banco de Dados da **Universidade Federal de Alagoas (UFAL) - Campus Arapiraca**.

## 📐 Modelo Conceitual
O Diagrama Entidade-Relacionamento (DER) foi desenhado seguindo a notação conceitual clássica (padrão de cardinalidades por limites mínimos e máximos). A imagem do diagrama encontra-se na pasta `/diagrama`.

## 🚀 Como Executar o Projeto
Os scripts SQL foram totalmente desenvolvidos e testados utilizando o **PostgreSQL**.

1. Execute o script de criação de tabelas localizado em `/sql/01_estrutura_banco.sql`.
2. Execute o script de povoamento com dados de teste em `/sql/02_povoamento_testes.sql`.
3. O arquivo `/sql/03_consultas_views.sql` contém as queries de relatórios complexos, joins, subconsultas e views prontas para validação.
